# PartySistersCards
聚会姐姐牌！打开之后可以点击按钮抽取扑克牌，牌下方文本框会提示玩家进行“姐姐牌”“逢七过”等多种玩法。并且该游戏删除了“小姐牌”而用“姐姐牌：直到下张2出现之前，你可以跟任何人一起喝酒，并且喝酒的人必须叫你‘姐姐’”代替，是十分女性友好的游戏。Party Sisters Cards! This design makes it a distinctly female-friendly drinking game. You can click a button to draw a playing card. A text box below the card will prompt players to engage in various drinking games.
